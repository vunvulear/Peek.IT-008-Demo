# Changelog

## 1.0.0 (2026-04-10)

- Initial release
- Add `/speckit.test.generate` command for generating test scaffolds from spec acceptance criteria
- Add `/speckit.test.coverage` command for mapping spec requirements to test coverage
- Add `/speckit.test.gaps` command for finding untested spec requirements
- Add `/speckit.test.plan` command for generating structured test plan documents
- Optional `after_implement` hook for scanning untested requirements
- Supports all major test frameworks and languages
